# Project for testing web site "Swag Labs"

## Setup

- Install Node.JS

- Install wdio

- Clone the project 

```bash
git clone https://github.com/BilikSergey/Testing-of-Swag-Labs.git
```

- Install dependencies:

```bash
npm install
```

#Run tests

To run a tests run the command below:

```bash
wdio run ./wdio.conf.js
```
